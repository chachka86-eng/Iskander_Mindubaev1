Random Password Generator
Автор: Миндубаев Искандер 

Описание
Приложение для генерации случайных паролей с графическим интерфейсом. Позволяет настраивать длину и состав пароля, сохраняет историю всех сгенерированных паролей в JSON файл.

Функции
- Генерация случайных паролей длиной от 4 до 32 символов
- Выбор типов символов: цифры, буквы, специальные символы
- Копирование пароля в буфер обмена
- Автоматическое сохранение истории в JSON
- Загрузка истории при запуске
- Очистка всей истории
- Экспорт истории в файл
- Проверка корректности ввода

Требования
- Python 3.6 или выше
- Стандартные библиотеки: tkinter, random, string, json, datetime, os


Использование

1. Установите длину пароля с помощью ползунка (4-32 символа)
2. Отметьте нужные типы символов:
   - Цифры (0-9)
   - Буквы (A-Z, a-z)
   - Специальные символы (!@#$%^&*)
3. Нажмите "Сгенерировать пароль"
4. Нажмите "Копировать" для сохранения пароля в буфер обмена
5. Все пароли автоматически сохраняются в таблице истории
6. Используйте кнопки "Очистить историю" и "Экспорт истории" для управления записями

Примеры использования

Пример 1: Простой пароль
- Длина: 8 символов
- Выбрать: только буквы
- Результат: "aBcDeFgH"

Пример 2: Надёжный пароль
- Длина: 16 символов
- Выбрать: цифры, буквы, спецсимволы
- Результат: "P@ssw0rd#2024!XyZ"

Пример 3: Цифровой пароль
- Длина: 6 символов
- Выбрать: только цифры
- Результат: "482915"
